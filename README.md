# 📊 Advanced EDA Project

## 🌟 Overview

This project focuses on **Advanced Exploratory Data Analysis (EDA)** . The goal of this project is to explore business data, generate meaningful insights, and create **clean summarized datasets** that can be easily used for **dashboarding and reporting**.

The analysis uses data from the **Gold Layer of a Data Warehouse**, which was **designed and built by me in my Data Warehouse project**. This ensures the data is already cleaned, structured, and optimized for analytics.

---

## 🏗️ Data Source

The analysis is based on the following tables from the **Gold Layer** of the Data Warehouse:

📦 **Fact Table**

* `sales`

📚 **Dimension Tables**

* `products`
* `customers`

These tables follow a **fact–dimension model**, which makes it easier to perform aggregations, segmentation, and performance analysis.

---

## 🔎 Advanced EDA Techniques Used

### 🧩 Part-to-Whole Analysis

This analysis helps understand how individual parts contribute to the overall result. For example, identifying how much each **product or category contributes to total sales**.

### 📈 Cumulative Analysis

Cumulative analysis tracks how values **build up over time**, helping to identify trends like **running sales totals and growth patterns**.

### ⏱️ Changeover Time Analysis

This analysis focuses on the **time gap between events**, such as the time between customer purchases or product sales.

### 🚀 Performance Analysis

Performance analysis evaluates how well **products or customers perform** based on metrics like sales, revenue, and order volume.

### 🧠 Data Segmentation

Segmentation groups **products or customers into meaningful categories** based on their behavior or performance, such as high-performing or low-performing groups.

---

## 📑 Summarized Reports

After completing the analysis, **summarized datasets** were created to make the data easier to use for **business intelligence, dashboards, and reporting**.

### 🛍️ Product Report

This report provides key insights about product performance, including:

* 📦 Total Orders
* 💰 Total Sales
* 🔢 Total Quantity Sold
* 👥 Number of Unique Customers
* ⏳ Product Lifespan
* 📅 Recency (months since last sale)
* 💵 Average Order Revenue (AOR)
* 📊 Average Monthly Revenue

---

### 👥 Customer Report

This report summarizes **customer purchasing behavior**, including:

* 🧾 Total Orders
* 💰 Total Sales / Total Spending
* 📅 Recency (time since last purchase)
* 💳 Average Order Value (AOV)
* 📊 Average Monthly Spending
